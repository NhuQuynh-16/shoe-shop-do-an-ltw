# Dự án Shop Bán Giày (Đồ án LTW)

Đây là dự án đồ án môn Lập trình web, xây dựng một trang web thương mại điện tử bán giày.

# Các chức năng chính
* Xem sản phẩm
* Giỏ hàng
* Đăng nhập / Đăng ký
* Thanh toán (mô phỏng)

# Công nghệ sử dụng
* **Back-end:** ASP.NET Framework (Web Forms / MVC 5)
* **Front-end:** HTML, CSS, JavaScript, JQuery
* **Database:** SQL Server
